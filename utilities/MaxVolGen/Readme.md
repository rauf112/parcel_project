# Maximum building volume generator
## Description
- This utility generates the maximum building volume for a given parcel in IFC 4x3 format. 
- It uses cadastral data and urban planning data to perform the generation.
- The urban planning parameters considered for the generation of the volume are:
  1. Maximum height
  2. Clearance
  3. Maximum buildable depth
  4. Building coefficient
  5. Maximum land occupancy
  6. Setbacks to boundaries and alignments  
- The IFC file generated is intended for architects and designers in the role of applicants who have to develop a building project in BIM located on a specific parcel of land in a municipality to obtain the building permit.
- The IFC file also includes other parameters that the applicants of the municipality should consider when developing their building project proposals.
- The applicant should also be able to download the volumetric data of the adjacent buildings in case they can affect the limits of its project (urban planning regulations govern the space between buildings).
